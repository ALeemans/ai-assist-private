# Schuifpui (Sliding Doors) Installation Project

## Overview
Installing sliding doors (schuifpui) at the back of our apartment. This requires coordination with VVE, notary changes, and potentially legal/administrative assistance.

## Goal
Replace current back doors with modern sliding doors (schuifpui) to improve access to outdoor space and modernize the apartment.

## Challenges
- **VVE Coordination**: Need approval from homeowners association
- **Notary Availability**: All notaries in Netherlands are very busy, hard to get appointments
- **Knowledge Gap**: Lack experience with this type of project, need expert assistance
- **Administrative Complexity**: Changes to property deed/documents required

## Key Stakeholders
- **VVE (Vereniging van Eigenaars)**: Homeowners association - approval needed
- **Notaris**: Legal changes to property documents
- **Vereniging Eigen Huis**: Can provide guidance and assistance for homeowners
- **Contractor**: For actual installation (when approved)

## Notary Requirements
- Possibly changes to splitsingsakte (apartment division deed)
- May need to update property boundaries or shared walls
- All VVE members may need to be informed/approve
- Costs can be significant (‚Ç¨500-2000+)

## VVE Approval Process
- Submit formal written request
- Provide technical drawings/plans
- Attend VVE meeting to present
- Get majority vote approval
- Receive written confirmation
- Ensure changes are documented in VVE records
