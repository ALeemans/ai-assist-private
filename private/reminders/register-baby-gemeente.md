# Register Baby at Gemeente

In Nederland kan je vanaf week 20 van de zwangerschap de geboorte van je baby alvast aanmelden bij de gemeente. Dit is handy om alvast voorbereid te zijn.

**Actions:**
- Check gemeente website for pre-registration process
- Prepare required documents (ID, proof of relationship)
- Schedule appointment if needed
- Understand what needs to be done after birth (you have 3 days to officially register after birth)

**Note**: Official registration must still be done within 3 days after birth, but pre-registration can make the process smoother.
